    AI-Driven Highlights Generator for Football Matches uses the YOLO algorithm to automatically identify key moments in both live and recorded football matches. The platform allows users to input live stream links or upload videos, instantly generating highlights. With an interactive frontend built in React, the system offers an intuitive user experience. This project provides football fans and content creators with an efficient way to relive key moments from matches, powered by advanced AI technology.
